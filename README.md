## How to play
- `WASD`移动
- `1~9`选择物品
- `左键`破坏方块
- `右键`放置方块
- `V`关闭UI

## Build and Run
### Download
```
git clone https://github.com/Ninter6/craftmine --recursive
cd craftmine
```
### Build
```
cmake -S . -B build
cmake --build build -j8
```
### Run
```
build/src/craftmine
```
